# Library webhooks

Library webhooks allows for easy receiving and parsing of Gitea, GitHub and GitLab Webhook Events

## Installation

```bash
go get -u github.com/ergoapi/webhooks
```

import package into your code

```go
import "github.com/ergoapi/webhooks"
```
